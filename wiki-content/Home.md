**A writing workflow for Obsidian.** Manage projects, chapters (optional), scenes, and versioned drafts in plain markdown, with flexible folder structure and native compatibility with Obsidian Bases.

Draft Bench is inspired by [Longform](https://github.com/kevboh/longform), with added emphasis on per-scene and per-chapter draft history as first-class files, rich metadata via frontmatter, and a compile system that requires no JavaScript knowledge.
