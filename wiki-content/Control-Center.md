# Control Center

The Control Center is Draft Bench's main hub — a tabbed modal where project overview, manuscript structure, templates, compile, and settings all live.

---

## Opening the Control Center

- Ribbon icon (look for the Draft Bench icon in the left sidebar).
- Command palette: **Draft Bench: Open Control Center**.
- Right-click a project folder or project note -> **Open Control Center**.

## Tabs

### Project

Project overview: title, metadata, synopsis, aggregate word count, status breakdown.

### Manuscript

Ordered scene list (sorted by `dbench-order`) with status and prior-draft count per scene. Click-through to open any scene.

A toolbar along the top surfaces primary project actions:

- **New scene**
- **New draft of current scene**
- **Reorder scenes** (opens the dedicated reorder modal)
- **Compile** (Phase 3+)

Ordering is read-only in the list — reorders happen in the focused modal.

### Templates

Manages the built-in scene template. User-defined templates and multi-template selection arrive in Phase 2.

### Compile

Book Builder interface. Phase 3+.

### Settings

Plugin configuration. See [Settings and Configuration](Settings-And-Configuration).

## Keyboard navigation

Draft Bench commits to keyboard-first operation. Expect arrow-key navigation between tabs, focus management across modal surfaces, and every action bindable via Obsidian's Hotkeys settings.
